# WeatherApp

Портфолио-проект для iOS, написанный на `Swift` и `UIKit`.

`WeatherApp` показывает текущую погоду, почасовой прогноз, прогноз на несколько дней, поиск городов и список избранных городов. Для получения данных о погоде и геокодинге приложение использует API [Open-Meteo](https://open-meteo.com/).

## Возможности

- Текущая погода для выбранного города
- Почасовой прогноз
- Прогноз на 7 дней
- Поиск города
- Избранные города
- Локальное сохранение избранного через `UserDefaults`

## Структура проекта

Проект разделен на несколько простых слоев:

- `App` - запуск приложения и настройка tab bar
- `Domain` - модели и протоколы приложения
- `Networking` - API-клиент, DTO и сервисы
- `Data` - репозитории и маппинг данных
- `Presentation` - экраны, ячейки и view model

Такой подход помогает отделить UI от сетевого слоя и упростить дальнейшее развитие проекта.

## API

В приложении используются:

- Open-Meteo Forecast API
- Open-Meteo Geocoding API

API key не требуется.
